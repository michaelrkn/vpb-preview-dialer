Virtual Phone Bank Preview Dialer

Thanks for using the Virtual Phone Bank Preview Dialer!

You can access this page at any time by clicking the "V" icon next to your address bar and choosing "Options".

Please give permission to use your microphone, so that you can make calls through the browser on Virtual Phone Bank.

Outgoing Caller ID

Below, enter the phone number you'd like to use for the outgoing caller ID below. If it's not already in our system, we'll call you to verify it.

Keyboard Commands

When you are on the phone, you can press "h" to hang up.

If you don't make a call at first, you can make it laster by pressing "c".

Uncheck the box below if you'd like to disable the pop-up that asks you if you'd like to call the number. You'll still be able to make calls by pressing "c".
