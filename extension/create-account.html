Virtual Phone Bank Preview Dialer

By Political Dialing Systems

You can access this page at any time by right-clicking the "V" icon next to your address bar and choosing Options.

Setup

Please give permission to use your microphone, so that you can make calls through the browser on Virtual Phone Bank.

Create a Campaign Code and Access Code for your account. These can be given to your team members to use the dialer. Then hit the "Twilio Connect" button to connect your Twilio account.

Options

Below, enter the phone number for the outgoing caller ID.
We'll call you to verify it if you haven't before.

Uncheck the box below if you'd like to disable the pop-up that asks you if you'd like to call the number. You'll still be able to make calls by pressing "c" or clicking the phone number.

Check the box below if you'd like your calls to ring until they reach a voicemail (30 seconds max). This will slow down your dialing considerably.
